Mods

Un mod es una instanciación del SDK para un activo, jurisdicción y circunstancia específicos. Reside en su propia rama (mods/{nombre}) y añade un directorio mod-{nombre}/ encima de la base. Cada mod demuestra que el protocolo no es una abstracción: es una herramienta que funciona con materia concreta — sea oro, sea moneda digital, sea lo que venga después.

Crear un mod nuevo

$ git checkout -b mods/{nombre}            # rama desde main
$ mkdir mod-{nombre}
$ cp sdk/config.template.md \
     mod-{nombre}/proyecto.config.md        # rellenar {{PLACEHOLDERS}}
# instanciar plantillas de roles, bitácoras, contrato
# crear referencia de cadena ética del activo
# escribir mod-{nombre}/README.md
→ instrucciones completas en README → config.template.md
